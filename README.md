Aplicação web que mostra informações climáticas em tempo real utilizando a **API OpenWeather**.

O usuário pode pesquisar qualquer cidade e visualizar dados como temperatura, descrição do clima, umidade e velocidade do vento.

Contras: O app funciona apenas com nomes de **Cidades** na sua versão 1.0

---

#  Funcionalidades

* Buscar clima de qualquer cidade
* Exibir temperatura atual
* Mostrar descrição do clima
* Mostrar umidade do ar
* Mostrar velocidade do vento
* Ícone representando o clima
* Interface simples e responsiva

---

#  Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* OpenWeather API
* Fetch API

---

#  Preview

Interface simples com campo de busca e um card exibindo as informações climáticas da cidade pesquisada.

#  Como usar

1. Clone o repositório

```
git clone https://github.com/andrealmeidao/ClimaxCity---Tempo.git
```

2. Abra o arquivo:

```
index.html
```

3. Digite o nome de uma cidade e clique em **Buscar**.
